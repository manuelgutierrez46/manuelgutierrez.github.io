1. Diferencias entre la etiqueta title y alt.

Title especifica información adicional sobre un elemento y alt se utiliza para aportar información sobre el elemento, en el caso de que este falle.

2. ¿Para que se utilizan los elementos map y area?

Map se usa para mapear una imagen, dando la posibilidad a poder incluir en ella, por ejemplo, varios enlaces, mientras que area define una sección en un mapa de imágenes.
